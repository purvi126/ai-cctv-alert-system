# AI CCTV Alert System

An AI-powered real-time CCTV safety alert system designed to detect accidents and fire incidents using computer vision. Built as part of Yuvamathon 2024 Hackathon.

## Tech Stack
- Python
- OpenCV
- NumPy
- Computer Vision
- Real-time Video Processing

## Features
- Real-time CCTV feed processing
- Accident detection logic
- Fire detection using color & motion analysis
- Alert triggering system
- Modular and extensible design

## Use Case
Designed for deployment in public areas, campuses, and smart city environments to improve safety and emergency response time.

## Status
Prototype completed and demonstrated at Yuvamathon 2024.
